== Description ==

This Plugin tracks user actions on posts, then displays recommendation, history, ranking on your WordPress.

"pageviews", "pages/visit", "avg. time on site" will grow up and "bounce rate", "abandonment rate" go down.

You need to apply "Submit! Recommend" in order to use this service.

Application form is below. (Japanese Only)
https://recommend.submit.ne.jp/apply/

== Installation ==

1. We check the applied URL whether it's possible to introduce or not.
2. Issue account id and password.
3. Enter account id and install plugin.
4. Start tracking on your WordPress posts.
5. Select area to display recommendation, history, ranking on your WordPress.
